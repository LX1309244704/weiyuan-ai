# 广告创作页面

这是一个基于Fabric.js的广告创作详情页，复刻了Nano Banana AI编辑器的功能和样式。

## 功能特性

### 🎨 创作模式
- **生成模式**: AI生成广告内容
- **编辑模式**: 手动编辑广告元素
- **遮罩模式**: 添加遮罩效果

### 🖼️ 画布功能
- **Fabric.js画布**: 支持文本、形状、图片等元素的编辑
- **缩放控制**: 支持画布缩放（10%-200%）
- **重置功能**: 一键重置画布内容
- **下载功能**: 导出PNG格式的广告图片

### 📝 提示词系统
- **智能提示**: 输入描述性提示词生成广告
- **高级设置**: 调节创意度和随机种子
- **参考图片**: 支持上传最多2张参考图片

### 📚 历史记录
- **变体管理**: 查看当前生成的广告变体
- **历史列表**: 保存最近的创作记录
- **快速下载**: 一键下载历史作品

## 技术栈

- **前端框架**: Next.js 14 + React 18
- **画布库**: Fabric.js 6.7.1
- **状态管理**: Zustand
- **UI组件**: Lucide React图标
- **样式**: Tailwind CSS

## 主要功能模块

### 左侧面板 (Prompt Composer)
- 创作模式切换（生成模式，编辑模式，遮罩模式）
- 参考图片上传
- 提示词输入框
- 高级参数设置
- 生成按钮

### 中间画布 (Canvas Area)
- Fabric.js画布
- 缩放控制工具栏
- 下载功能
- 空状态提示

### 右侧面板 (History)
- 当前变体展示
- 历史记录列表
- 快速下载功能

## 使用说明

1. **登录认证**: 需要登录后才能访问创作页面
2. **选择模式**: 根据需求选择生成/编辑/遮罩模式
3. **上传参考**: 可选上传参考图片（最多2张）
4. **输入提示**: 详细描述想要生成的广告内容
5. **调节参数**: 设置创意度和随机种子
6. **生成广告**: 点击生成按钮创建广告
7. **编辑调整**: 在画布上直接编辑元素
8. **下载保存**: 导出最终的广告图片

## 开发说明

### 依赖安装
确保已安装以下依赖：
```bash
npm install fabric@^6.7.1
```

### 本地开发
```bash
npm run dev
```

### 构建部署
```bash
npm run build
npm start
```
